# Smart Fish Feeder IoT - Dokumentasi

Folder ini berisi dokumentasi teknis untuk proyek Smart Fish Feeder IoT.

## Daftar Dokumen

1. [Product Requirements Document (PRD)](file:///h:/Folder%20Hanpi/AppAdmin/fish-feeder/docs/PRD.md)
   - Berisi scope MVP, arsitektur sistem, desain database, user flow, rancangan antarmuka, dan sukses metrik.
2. [Supabase REST API Reference for ESP32](file:///h:/Folder%20Hanpi/AppAdmin/fish-feeder/docs/API.md)
   - Dokumentasi lengkap mengenai endpoint, payload, dan integrasi yang digunakan oleh firmware ESP32 untuk berkomunikasi dengan Supabase.
3. [Skema Database Migration (SQL)](file:///h:/Folder%20Hanpi/AppAdmin/fish-feeder/docs/migration.sql)
   - Script SQL lengkap yang digunakan untuk menginisialisasi database di Supabase (tabel, kebijakan Row Level Security, auto-update trigger, dan channel Realtime).
